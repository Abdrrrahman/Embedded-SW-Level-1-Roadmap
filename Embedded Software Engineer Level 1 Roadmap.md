# Recommended Learning Path Timeline
**Total Duration: 9-13 months**
1. **Phase 1**: [[1. Software Fundamentals]] (2-3 Months)
	1. C Programming Language and Embedded C Topics
	2. Data Structures and Algorithms
	
2. **Phase 2**: [[2. Hardware Fundamentals]] (2-3 Months)
	1. Basic Logic Design
	2. Computer Organization and Architecture
	
3. **Phase 3**: [[3. Build Tools]] (1 Month)
	1. Make and CMake
	2. GDB andGCC
	
4. **Phase 4**: [[4. AVR Interfacing]] (2-3 Months)
	1. Microcontrollers
	2. AVR Interfacing
	3. Basic Protocols
	
5. **Phase 5**: [[5. RTOS Basics]] (1 Month)
	1. Bare Metal VS RTOS
	2. FreeRTOS Fundamentals
	
6. **Phase 6**: [[6. Software Testing Basics]] (1 Month)
	1. Unit Testing
	2. Integration Testing
	3. Mocking & Stubbing
	4. Test-Driven Development
	5. Code Coverage, Assertions and Defensive Programming
	
7. **Phase 7**: Level 1 Graduation Project (1 Month)
